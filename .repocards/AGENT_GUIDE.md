## How to use the graph

`graph.json` has three top-level maps:
- `files` — per-file metadata, symbols, and raw import list
- `imports_in[file]` — array of files that import `file`
- `imports_out[file]` — array of files `file` imports (resolved to local paths)

For "who uses X?" look up `imports_in`. For "what does X depend on?" use `imports_out`. Combine with `index.json` to map symbol names to files first.
